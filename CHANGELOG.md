# Changelog

All notable changes to this project will be documented in this file.

## [v1.2.0] - 6-NOV-2025
### Added
- Initial public release of the APK Addon for World of Warcraft.
- Includes core files: APK.lua, APK_UI.lua, and APK.toc.
- Supports auto-pet summoning, favorites, and blacklist features.

### Fixed
- Minimap button placement and UI checkbox alignment.
- Version number synchronization in .toc file.

### Notes
This release establishes the full repository link between the live AddOns folder and GitHub.
Future versions will include incremental improvements and bug fixes.
